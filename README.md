# Personal Budget App

A modern personal finance management application built with React Native and Expo. Track your income, expenses, and manage multiple wallets with a clean, intuitive interface.

## Features

- **Multi-wallet management** - Handle different accounts (checking, savings, credit cards)
- **Transaction tracking** - Record income, expenses, and transfers
- **Real-time balance updates** - Monitor your net worth
- **Category organization** - Organize transactions for better insights
- **Offline-first** - Local SQLite database for reliable data storage
- **Cross-platform** - Works on iOS and Android

## Tech Stack

- React Native 0.79.5
- Expo SDK 53
- React Query (TanStack Query)
- SQLite with Expo SQLite
- React Navigation v7
- Custom UI component library

## Getting Started

### Prerequisites

- Node.js (v18+)
- npm or yarn
- Expo CLI
- iOS Simulator or Android Studio

### Installation

1. Clone the repository
   ```bash
   git clone <repository-url>
   cd personal-budget
   ```

2. Install dependencies
   ```bash
   npm install
   ```

3. Start development server
   ```bash
   npm start
   ```

4. Run on device/simulator
   - Press `i` for iOS
   - Press `a` for Android
   - Scan QR code with Expo Go app
